# B2
Displacement = 26.5 $\pm$ 0.5m
Time = 0.6s

Work out Velocity
$$V = S/t = 26.5 / 0.6 = 41\frac{1}{6}$$

$$UC = S + t = \frac{0.5}{26.5} + \frac{0.1}{0.6} = 0.185$$
$$UC = 0.185 * 41\frac{1}{6} = 7.637 \ldots$$

$$V = 41 \pm 7m/s$$
